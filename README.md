# Sistem Katalog Produk Berbasis Web

## Deskripsi Proyek
Sistem Katalog Produk Berbasis Web ini dikembangkan untuk **mengoptimalkan digitalisasi produk batik** melalui integrasi dengan marketplace eksternal. Sistem ini memungkinkan pelaku usaha batik untuk:

- Menampilkan katalog produk secara online.
- Mengelola informasi produk seperti nama, deskripsi, harga, dan stok.
- Menghubungkan produk dengan platform marketplace eksternal untuk memperluas jangkauan penjualan.
- Mempermudah pelanggan dalam melihat dan membeli produk batik secara digital.

Proyek ini bertujuan **mendukung transformasi digital UMKM batik** dan meningkatkan efisiensi pemasaran serta penjualan.

## Fitur Utama
1. **Manajemen Produk**
   - Tambah, edit, dan hapus produk.
   - Kelola kategori dan atribut produk.
2. **Integrasi Marketplace**
   - Sinkronisasi produk dengan marketplace populer.
   - Otomatis update stok dan harga.
3. **Pencarian & Filter**
   - Fitur pencarian produk berdasarkan nama atau kategori.
   - Filter produk berdasarkan harga, kategori, atau popularitas.
4. **Dashboard Admin**
   - Ringkasan statistik penjualan dan produk.
   - Kontrol penuh terhadap katalog dan integrasi marketplace.
5. **Responsif & User-Friendly**
   - Tampilan website dapat diakses dengan baik melalui desktop maupun perangkat mobile.

## Teknologi yang Digunakan
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** PHP
- **Database:** MySQL / MariaDB
- **Integrasi Marketplace:** API marketplace eksternal (Tokopedia, Shopee, dan Lazada)
- **Version Control:** Git
